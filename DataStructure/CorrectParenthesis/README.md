


<h2>올바른괄호(Stack)</h4>

<h3>-설명</h3>

괄호가 입력되면 올바른 괄호이면 "YES", 올바르지 않으면 "NO"를 출력.<br/>
(())() 이것은 괄호의 쌍이 올바르게 위치한것이므로 "YES"가 출력.<br/>
(()()))은 올바른 괄호가 아니므로 "NO"가 출력.

<br/><br/>

<h3>-입력</h3><br/>

첫 줄에는 괄호 문자열이 입력된다. 문자열의 최대 길이는 30.
<br/>
<br/>

<h3>-출력</h3><br/>
첫 번째 줄에 YES,NO를 출력<br/>

<br/>

예시입력 1) 
```java
(()(()))(()
```
</h4>
 
예시출력 1) 
<br/>

```java
NO
```




<h2> 부분집합 구하기 </h2>

<h3>-설명</h3>

자연수 N이 주어지면 1부터 N까지의 원소를 갖는 집합의 <br/>
부분 집합을 모두 출력해보자.

<br/><br/>

<h3>-입력</h3><br/>

첫번째 줄에 자연수 N(1<=N<=10)이 입력된다
<br/>

<br/>

<h3>-출력</h3><br/>
첫 번째 줄부터 각 줄에 하나씩 부분집합을 예시출력과 같은 순서로 출력한다.<br/>
단 공집합은 출력하지 않는다.

<br/>
<br/>

예시입력 1) 
```java
3
```
</h4>
 
예시출력 1) 
<br/>

```java
1 2 3
1 2
1 3
1
2 3
2
3
```
<br/>

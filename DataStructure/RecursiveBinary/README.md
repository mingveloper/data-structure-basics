


<h2>재귀함수를 이용한 이진수 출력 (Recursive)</h4>

<h3>-설명</h3>

10진수 N이 입력되면 2진수로 변환하여 출력할것. (단, 재귀함수를 이용할 것)<br/>

<br/><br/>

<h3>-입력</h3><br/>

첫 번째 줄에 10진수 N(1<=N<=1,000)이 입력 됨.
<br/>

<br/>

<h3>-출력</h3><br/>
첫 번째 줄에 이진수를 출력.<br/>

<br/>

예시입력 1) 
```java
11
```
</h4>
 
예시출력 1) 
<br/>

```java
1011
```
